Pintos Operating System

This is an open source project based off the Pintos operating system created by Ben Pfaff at Standford Universtiy. Within this project I worked in a group of 3 to further implement features to the operating system that include, alarm clock for threads, a priority scheduler, and an advanced scheduler including round robin scheduling.
